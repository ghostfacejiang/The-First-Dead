# The-First-Dead

----------------------------------------------------------------
## 游戏定位
----------------------------------------------------------------

* rougelike
* 自动战斗
* 卡牌策略
* 3D俯视角
* 固定镜头

开发引擎：
* Godot 4.4.1
* GDScript

----------------------------------------------------------------
## 核心机制
----------------------------------------------------------------

主角通过操作死亡获取力量。

而死亡会产生尸体，尸体进入墓地。

墓地会成为资源为主角所用。

主角不断不断使用墓地的力量会提高腐化值。

腐化会使世界逐渐崩坏，敌人将越来越强。

最终主角发现Boss与自己拥有相同能力。

自己正在成为曾经被自己讨伐的存在。

----------------------------------------------------------------
## 核心循环
----------------------------------------------------------------

1. 场上单位死亡，产生尸体留在场上
2. 主角可选择复活、吞噬或引发尸爆
3. 结算后尸体进入墓地成为主角的资源
4. 主角可以消耗墓地资源召唤亡灵或强化随从
5. 腐化值将随着操作墓地的次数而提高
6. 敌人将得到特殊强化
7. 杀死敌人或者被敌人杀死
8. 重复循环

----------------------------------------------------------------
## 局内设计
----------------------------------------------------------------

### 战场
* 场上共有12个格子，敌我方各6个
* 格子划分为前排3个后排3个

### 战斗
* 场上单位自动攻击
* 主角通过操作卡牌、尸体、墓地来战斗
* 卡牌可以强化友方、释放法术、攻击对方

### 资源

#### 精力
最大值为10，每秒恢复1点，用于打出卡牌

#### 魂魄
通过吞噬尸体获得，用于召唤亡灵

#### 墓地
用于记录死亡单位，有记录上限，可以跨战斗继承

#### 腐化
使用禁忌能力获得，决定世界变异程度

----------------------------------------------------------------
## 初始卡牌
----------------------------------------------------------------

### 复苏
消耗3精力，选择尸体复活，腐化+1

### 吞噬
消耗1精力，选择尸体进入墓地，魂魄+1，腐化+1

### 尸爆
消耗2精力，摧毁尸体造成范围伤害，腐化+1

### 招魂
消耗5魂魄，召唤骷髅军团，腐化+2

----------------------------------------------------------------
## 腐化阶段
----------------------------------------------------------------

### 腐化10
敌人生命+10%

### 腐化20
出现新敌人“神父”，可以净化尸体

### 腐化30
敌人将会利用尸体
